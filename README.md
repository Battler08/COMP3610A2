## Project Structure

```
assignment2.ipynb   # Main notebook
data/
  raw/              # Raw downloaded data
  cleaned_data.parquet  # Cleaned dataset
requirements.txt    # Dependencies
README.md
```

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

Key libraries: `polars`, `scikit-learn`, `torch`, `matplotlib`, `seaborn`, `pandas`, `numpy`

## How to Run

1. Clone the repository
2. Install requirements
3. Run all cells in `assignment2.ipynb` from top to bottom
4. Data will be downloaded automatically on first run
